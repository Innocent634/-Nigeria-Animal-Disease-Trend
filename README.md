# -Nigeria-Animal-Disease-Trend
## Project Objective To analyze Animal Disease data across selected states in Nigeria providing useful details regarding disease outbreak for monitoring and containment.
## Dataset Used

## Questions
- Which disease resulted to the highest death?
- Which diseases have the highest severity index?
- What region recorded the highest disease outbreak?

## Key Points Covered
- Regional disease outbreak distribution
- Impact of disease type on total animal deaths
- Animal disease severity index
- Yearly disease outbreak trend 

## Dashboard

### Data Preparation
- Animal Disease Trends AI generated  raw data table was used
  
### Visual Design:

- Bar/Column Charts for comparison (Disease, Region)
-	Line Chart with Forecast for trends 
-	Slicer for filtering by State, Region, Year and Disease Category 
-	Cards for KPIs (Total Animal Outbreak, Affected, Deaths and Vaccinated) 

### Conclusion:
This analysis on animal disease trends offers viable information to government authorities and relevant stakeholders to:
- Identify prevalent animal disease 
- Understand impact of disease on the animal production market 
- Make data-driven decisions geared towards disease control and prevention
- Predict future outbreak trends based on sustained current management techniques
